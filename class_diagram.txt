note bottom of MQTTService
  Protocol: Wi-Fi + MQTT over TCP
  Topics:
   sensors/data — ESP32 → App
   cmd/pump     — App → ESP32
   status/pump  — ESP32 → App
end note

note bottom of DecisionEngine
  Strategie de decizie:
  1. Se încearcă AIDecisionEngine
  2. Dacă isAvailable() = false
     → se folosește RuleBasedEngine
  3. În mod manual, decizia aparține
     exclusiv utilizatorului
end note
